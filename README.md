# Text-Editor

# Concepts Used:

* Python Tkinter package for GUI
* Function Overloading in python
* Event & Exception Handling 
* File Handling 

# Installation:

1. Clone Respository

    ```makefile
    $ git clone https://github.com/Nishant-Badgujar18/Text-Editor.git
    ```

2. Install Tkinter

    ```makefile
    $ sudo apt install python3-tk
    ```
    
# Run:

-   To Run Text Editor

    ```makefile
    $ python3 Text-Editor.py
    ```
